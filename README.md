# 🍫 Awesome Chocolates Sales Dashboard – Power BI Project

## 📌 Project Overview
The **Awesome Chocolates Sales Dashboard** is an interactive Power BI dashboard built to analyze and visualize the overall business performance of a chocolate company.

The dashboard provides insights into:
- Sales Performance
- Profitability
- Shipment Trends
- Product Analysis
- Salesperson Performance
- Regional Insights

A unique feature of this project is the use of **Bookmarks and Navigation Icons** to dynamically switch visuals within the **same dashboard page**, creating a smooth and interactive user experience without using multiple report pages.

## Interactive Bookmark Navigation
This dashboard uses **Bookmarks** and **Buttons/Icons** to switch between different visual states within the **same report page**.

### Example Interactions:
- Switching between:
  - 👨‍💼 Salesperson Analysis
  - 🍫 Product Analysis
- Dynamic visual transitions using side navigation icons

This approach improves:
- User experience
- Dashboard interactivity
- Navigation efficiency
- Overall report design

# 🔖 Bookmark Functionality

Instead of creating multiple pages, bookmarks are used to create different dashboard views dynamically within a single page.

### Benefits of Using Bookmarks:
- Cleaner dashboard layout
- Faster navigation
- Better storytelling
- Improved interactivity
- App-like dashboard experience
